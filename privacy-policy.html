Политика конфиденциальности 4Pulse

4Pulse — браузерное расширение для работы с форумом 4PDA.

Какие данные используются

Расширение может обрабатывать данные авторизованной сессии 4PDA, сообщения QMS, уведомления, избранные темы, упоминания, тикеты и содержимое страниц форума, необходимое для работы функций расширения.

Cookies и авторизация

Cookies используются только для определения состояния авторизации пользователя на 4pda.to и получения персонализированных данных аккаунта. Cookies не копируются, не продаются и не передаются третьим лицам.

Хранение данных

Настройки расширения, параметры интерфейса, состояние радио, кеш счётчиков и временные служебные данные сохраняются локально средствами браузера.

Передача данных

4Pulse не продаёт пользовательские данные и не передаёт их третьим лицам. Данные используются только для работы функций расширения.

Удалённый код

Расширение не загружает и не выполняет удалённый JavaScript или WebAssembly-код.

Внешние сервисы

Для встроенного радио могут использоваться внешние сервисы поиска радиостанций, получения метаданных и загрузки шрифтов. Эти запросы не используются для рекламного профилирования пользователей.

Контакты

Вопросы по политике конфиденциальности можно направлять через репозиторий проекта 4Pulse на GitHub.
